# NailBiteMenu
A real-time menu bar app that detects nail-biting using your Mac’s camera.

**Download:** [NailBiteMenu.dmg](https://github.com/yourname/NailBiteMenu/releases/latest/download/NailBiteMenu.dmg)

#### How to install
1. Open the DMG and drag NailBiteMenu into Applications.
2. If macOS blocks it, go to **Settings → Privacy & Security → Open Anyway**.
3. After launch, it runs quietly in your menu bar.

#### About
NailBiteMenu uses on-device camera processing: nothing leaves your Mac!
